Advanced CSS Selectors (Simple)

Learn some useful CSS selectors with easy examples you can use in your projects.

1. Attribute Selectors

Style elements based on attributes, without adding classes.

  • [type="text"]: Select all inputs with type="text".
  • [title^="Start"]: Select elements whose title starts with "Start".
  • [class$="end"]: Select elements whose class ends with "end".
  • [alt*="image"]: Select elements whose alt attribute contains "image".

2. Pseudo-classes

Target elements based on position or state.

  • :not(.skip): Select elements without class "skip".
  • :empty: Select elements with no content.
  • :nth-child(2): Select the second child element.
