CSS Box Model

The CSS Box Model is a fundamental concept describing the rectangular boxes generated for elements in the document tree. Each box consists of four areas: content, padding, border, and margin. Understanding how these layers interact helps control layout and spacing effectively.

Example: Default Box Model (content-box)

Content-box box model
Width = 200px (content only)
Padding = 20px
Border = 5px solid blue
Margin = 30px

Border-box box model
