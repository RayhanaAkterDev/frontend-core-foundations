CSS Overflow

The CSS overflow property controls how content that exceeds the dimensions of a container is displayed. It defines whether content is clipped, hidden, or scrollable.

Overflow: visible

This is the default behavior where content spills out of the container.

This box has more text than it can contain, so the overflow is visible and spills outside.

Overflow: hidden

The overflowing content is clipped and hidden from view.

This box clips the overflowing content and hides the excess text. The text is long enough to overflow the box.

Overflow: scroll

Scrollbars are always visible, allowing access to all content.

This box has so much text that it will overflow its fixed size and scrollbars will appear allowing you to scroll and see all the content inside this container. Keep adding enough text to ensure overflow happens.

Overflow: auto

Scrollbars appear only if the content is larger than the container.

This box shows scrollbars only when the content exceeds container size. The content here is also long enough to overflow and show scrollbars.
