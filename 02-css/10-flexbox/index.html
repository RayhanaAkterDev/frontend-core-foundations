CSS Flexbox

CSS Flexible Box Layout (Flexbox) is a one-dimensional layout method for arranging items in rows or columns. It helps to distribute space dynamically and align items inside a container even when their sizes are unknown or dynamic. Flexbox excels at building flexible and responsive layouts quickly.
