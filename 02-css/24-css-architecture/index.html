CSS Architecture & Methodologies

CSS architecture is about organizing your styles in a clean and scalable way. These methodologies are used in real projects to avoid confusion, bugs, and class conflicts.

1. BEM (Block Element Modifier)

BEM is a class naming system to make your CSS readable and reusable.

  • Block: Main component (e.g., card)
  • Element: Part of the block (e.g., card__title)
  • Modifier: State or variation (e.g., card__title--active)

Example:

<div class="card">
  <h2 class="card__title card__title--active">Blog Post</h2>
  <p class="card__description">BEM keeps your CSS organized.</p>
</div>

2. SMACSS (Scalable and Modular Architecture for CSS)

SMACSS groups styles by their purpose. It helps teams write organized and scalable CSS.

  • Base: Reset, typography, default styles
  • Layout: Main sections like header, footer
  • Module: Reusable parts like buttons or cards
  • State: Temporary changes like .is-hidden
  • Theme: Styling variations like light/dark

Example:

<button class="btn is-active">Click Me</button>

3. OOCSS (Object-Oriented CSS)

OOCSS treats styles like objects to reuse layout patterns easily.

  • Split structure and visual styles
  • Separate content from container
  • Helps reduce code duplication

Example:

<div class="media">
  <img src="avatar.jpg" class="media__img" />
  <div class="media__body">
    <p>This is an object-style reusable layout.</p>
  </div>
</div>

4. Atomic CSS

Atomic CSS uses utility classes to apply styles directly in HTML.

  • Each class does one thing (e.g., .text-center)
  • Fast to write and lightweight
  • Popular in frameworks like Tailwind CSS

Example:

<div class="bg-blue-500 text-white p-4 rounded">
  Utility-first styling using Atomic CSS
</div>

5. CSS-in-JS

In component-based libraries like React, CSS can be written inside JS files using libraries like styled-components.

  • Styles are scoped to components
  • Supports dynamic styling with JS logic
  • Common in modern frontend development

Example:

// React + styled-components
const Button = styled.button`
  background: teal;
  color: white;
  padding: 10px;
  border-radius: 6px;
`;
