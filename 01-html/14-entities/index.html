HTML Entities

Some characters have special meanings in HTML. To display these characters correctly on the page, we use HTML entities.

Common Reserved Characters

  • & → &
  • < → <
  • > → >
  • " → "
  • ' → '

Special Symbols

  • © → ©
  • ® → ®
  • € → €
  • ¥ → ¥
  • £ → £

Numeric Character References

  • © → © (©)
  • ® → ® (®)
  • € → € (€)
  • " → " (")
  • < → < (<)
