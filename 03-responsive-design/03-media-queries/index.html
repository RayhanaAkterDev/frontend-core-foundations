Media Queries in CSS

Media queries allow us to apply CSS rules conditionally based on the characteristics of the user‚Äôs device, such as width, height, resolution, and orientation.

üîß Basic Syntax

The most common format for a media query is:


@media (condition) {
  /* CSS rules */
}

Example using screen width:


@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

üìê Min-width vs Max-width

  • min-width: Apply styles when the viewport is greater than or equal to a value (used in mobile-first design).
  • max-width: Apply styles when the viewport is less than or equal to a value (used in desktop-first design).
