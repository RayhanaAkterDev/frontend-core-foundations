Responsive Typography Example

এই উদাহরণে আমরা দেখব কিভাবে বিভিন্ন CSS টেকনিক ব্যবহার করে ফন্ট সাইজ স্ক্রীনের আকার অনুযায়ী পরিবর্তন করা যায়, যাতে পড়া সহজ হয় এবং ডিজাইন সুন্দর থাকে।

Fluid Font using clamp()

এই প্যারাগ্রাফের ফন্ট সাইজ `clamp()` ব্যবহার করে দেয়া হয়েছে, যা ছোট থেকে বড় স্ক্রীন পর্যন্ত স্বয়ংক্রিয়ভাবে অ্যাডজাস্ট হয়।

Relative Units (em, rem, vw)

রিলেটিভ ইউনিট ব্যবহার করলে টেক্সট সহজে স্কেল হয় এবং ব্যবহারকারীর ব্রাউজার সেটিংস অনুসারে অ্যাডজাস্ট হয়।
